## 📌 Project Overview

This project analyzes factory-to-customer shipping route efficiency
for Nassau Candy Distributor across the United States and Canada.
The goal is to identify efficient routes, detect geographic
bottlenecks, and provide data-driven logistics recommendations.

---

## 🎯 Problem Statement

Nassau Candy Distributor lacked visibility on:
- Which factory-to-customer routes are consistently efficient
- Which routes experience frequent delays
- How shipping performance varies by region, state, and ship mode
- Where operational bottlenecks exist geographically

---

## 📊 Key Findings

| Metric | Value |
|--------|-------|

| Best Route | Secret Factory → New Mexico (Score: 1.0) |
